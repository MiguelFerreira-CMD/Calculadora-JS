Calculadora em JavaScript

Uma calculadora simples desenvolvida utilizando HTML, CSS e JavaScript, capaz de realizar operações matemáticas básicas de forma rápida e intuitiva.

----

Adição;
Subtração;
Multiplicação;
Divisão;
Limpar cálculo.

----

Interface simples e fácil de usar!

Tecnologias utilizadas:

HTML5;
CSS3;
JavaScript.
